## Install

Via Composer

``` bash
$ composer require draperstudio/eloquent-repositories
```

And then include the service provider within `app/config/app.php`.

``` php
'providers' => [
    DraperStudio\Eloquent\Repositories\ServiceProvider::class
];
```
